# Email Template

General Email Templating Engine.

## Installation

```bash
yarn / npm install / npm i
```

## Usage
Structure:

```python
src 
-- layout
   -- layout.njk
-- modules
   -- header.njk
   -- footer.njk
-- templates
   -- template.njk

```
## Run project
```bash
gulp // to watch changes in template
```
```bash
yarn start // to run node server
```

## Build project
```bash
gulp build
```
